The World Wide Web and Internet

The Internet is probably one of the most important things in our lives today. Originally the internet was a Department of Defense project. They wanted ways of transferring data without using telephone lines as they were prone to damage. In February of 1958, Neil McElroy of the Department of Defense signed the Dept. of Defense Directive 5105.15 which launched the Advanced Research Project Agency(ARPA). This was the start of the internet. ARPANET was the first working internet model that sent its first message on 8/29/1969. One computer was located at UCLA and the other at Stanford University. 2014 marketed 45 years since ARPANET was invented.

This man is Time Berners-Lee, the father of the World Wide Web. One misconception is that the Internet and Web are the same thing, but they aren't. The Web is the way people acccess data in the form of websites and hyperlinks. Tim Berners-Lee worked at CERN. Lee also developed HTTP, Hyper-Text Transfer Protocol.


Google came along in 1998. This is Google's orginal homepage.

These two guys are Vinton Cerf (left) and Rober Kahn (right). They developed the Transmission Control Panel and Internet Protocol for the internet, also known as TCP/IP. This was a model that set the standards of data transission between networks. ARPANET adopted this model in the beginning of 1983.

The first web browser was called the Mosaic web browser In 1993, NCSA (Uni. of Illinois) developed this browser. This browser was the first to offer a GUI. It had a url bar, back button and forward button. It also was the first to display images along with text.

Binary Logic, Logic Gates, Memory, and Storage

Modern computing relies solely on Binary Digits 1 and 0. Binary Digits represent a form of math called Boolean Logic which means there are two possible conditions, TRUE or FALSE. These are furthermore represented by HIGH or LOW voltages. Boolean Logic is also very reliable when it comes to storing data. There are many different sets of encoding rules when Binary Digits are used. Binary digits can represent a lot. When you want to represent text you use ASCII. Every pixel in an image is represented by 3 binary sequences that correspond to a primary color. Numbers are used to determine the intensity of the color, then the numbers are transmitted to the liquid crystals. If multiple colors reoccur in a row, the data may be compressed to save space. Sound waves are stored via Pulse-Code Modulation. The Computers records the Amplitude of the sound wave every few seconds and stores the values as binary digits. There could be as many as 44,000 binary digits in one second of sound. When played back, the digits tell the speakers how the coils should vibrate to produce sound.


To go deeper, when a bit is HIGH or ON, it means that it is also TRUE.


Logic Gates are the equations implemented within a Logic Gate Circut. Logic Gates Produce a 1 or 0 based on input. Logic Gates can be used to do more complicated functions such as addition or subtraction. As gates get more complex, there can be more than 2 inputs. Below are four basic Logic Gates with their truth tables and symbols. A truth table is used to organize inputs and outputs.



In an AND Gate, when both or all inputs are 1, then the output is 1.



In an OR Gate, when at least one input is 1 then the output will be 1.



Many may get confused when it comes to memory and storage. Many would say they are the same, but there are differences. Inside of a computer, memory is typically static or temporary, meaning if a computer loses power, then the data in the memory gets lost. Below is a unit of memory chart. You can figure out how many bits are in a Megabyte, or how many Megabytes are in a Gigabyte. This specific chart allows you to go up to a Petabyte of storage. Many personal computers have an average of 1 Terabyte of storage, so a Petabyte would be 1000 Terabytes. Unless you are a photographer, make videos, or do anything that requires a large amount of space, 1 Terabyte is usually enough for the average person.


0 is the opposite, when the bit is LOW or OFF, it means that it is also FALSE.

Above is a chart that shows certain colors. Each color has three corresponding byte values next to them that go from 0-255. The three bytes represent the three primary colors, Red, Blue, and Green. It starts with red, then blue, then green, (R,B,G). so for example a number with a (255,0,0) would be red. A color with (200,0,255) would be a blue-purple color.



NOT Gate makes the output the oppsite of the input. The bar over the A means NOT, so the output is NOT A.

Common Programming Languages

Programming Languages are sets of instructions computers follow to perform tasks. For example, the browser running this program is following instructions to display this infographic correctly. When you click on an image in this infographic The computer follows more instructions to open up the page for more information. There are many types of programming languages such as object-oriented, Functional programming, procedural, etc. Object-Oriented means that the data within the program stored through objects. This also allows the specific data to be used more than once throughout the code. Procedural Programming means that the language uses things such as loops and many variables that eventually lead to a result. A basic way to tell languages apart is to see if they are compiled or not. Another way professionals tell languages apart is by the way you write the code. When code is written in a way that tells a computer how to complete a task, it is an imperative language. When code is written by the programmer that tells the computer exactly what to do, the language type is declarative.

Interpreted Languages and Markup Languages

An interpreted language is a programming language that doesn't need to be compiled. To compile in programming means to turn into machine or assembly code. This code can be read directly by the processor.


Above is the Javascript programming language, used in many websites. Javascript is an object-oriented scripting language, which means that data and functions is stored in the form of objects. Javascript is a scripting object-oriented language because it carries traits of a full object-oriented language, but it doesn't carry specific traits to make it a full programming language. With Javascript, you can also get libraries for Javascript that allow it to run outside of a browser. One popular library in Node.js. With Node, you can program many things such as whole webservers.



HTML stands for Hyper-Text Markup Language. It is used to make websites for people to view. It is used directly with Javascript and CSS to make a website better. For example with just HTML, websites would be very boring.


Compiled Languages

A compiled language is a programming language that has to be compiled into machine or assembly language. These languages usually are harder to learn because their concept is harder.


You may be thinking, Java and Javascript? sounds like the same thing, but they are very different. Javascript helps add functionality to websites and it is only mainly run in a browser. Java can be also used on the web, but serverside, meaning it deals with the server portion of a website. Java can do a lot. You can program apps for android and desktops. The other main difference is that Java has to be compiled and Javascript doesn't. They are even different when it comes to the type of language they are. Java is object-oriented and Javascript is a scripting object-oriented language.


C++

C++ (C Plus, Plus) is another compiled programming language. It is used to build may applications and games. For example, Microsoft Word is built using C++. C++ is considered to be a multi-paradigm language meaning it can be object-oriented, procedural, and a functional programming language.

The GUI (Graphic User Interface) and The Internet

The Graphic User Interface was a huge advancement in computing. Instead of writing instructions in a console to your computer, you could now click on icons to open certain programs. In the 1970's Xerox Labs began building a systems like the GUI, which later turned into the GUI. Before the GUI, you would have to type commands into a console and then the computer would follow the instructions. For example if you wanted to save to a text file you may have to use the command 'open [filename].txt, then edit it in the console. With the GUI, you could just click the file and edit it in a program, such as notepad. Parts of the GUI that are common are buttons, cursors, and icons you can click on.

The two shown above are Steve Wozniak and Steve Jobs, they were the first to revolutionize the GUI in their 1984 Apple Mac.


Above is a computer mouse. It may be odd to think, but this common computer Input/Output device hasn't always existed with computers. Douglas Engelbart invented this device in 1964. It was orginally made out of wood, a circut board, and a wheel that when it was placed on a desk would control the device. In 1972, Bill English invented innovated the mouse with a ball. English worked at Xerox, which was one of the first to use the GUI in a computer. The mouse made it easier for the user to control what happened on the computer. In 1988 Lisa M. Williams and Robert S. Cherry invented the optical mouse.
